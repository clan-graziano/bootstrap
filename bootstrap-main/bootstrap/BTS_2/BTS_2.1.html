Domande sull'Exercise BTS_2.1

1. Qual è la dimensione del testo di default di Bootstrap?

La dimensione del testo di default in Bootstrap è di 1rem, che corrisponde a 16px in base al font-size predefinito del browser.

2. Qual è il font di default di Bootstrap?

Il font di default in Bootstrap è Helvetica Neue, seguito da Helvetica, Arial e sans-serif.

3. Quali sono il margine superiore e inferiore di un paragrafo?

Il margine superiore e inferiore di un paragrafo in Bootstrap è di 1rem, che corrisponde a 16px.

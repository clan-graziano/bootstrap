A cosa serve Bootstrap?

Bootstrap è un framework CSS open-source progettato per facilitare la creazione di siti web e applicazioni web.

Fornisce strumenti e componenti predefiniti, come griglie, bottoni, moduli e navigazione, rendendo più semplice e veloce lo sviluppo di interfacce responsive e attraenti.

Cosa devi importare per creare una pagina web con Bootstrap?

Per utilizzare Bootstrap in una pagina web, devi importare il file CSS e il file JavaScript. Puoi farlo includendo i seguenti link nel tuo file HTML:

                        <link rel="stylesheet" href="https://stackpath.bootstrapcdn.com/bootstrap/5.3.0/css/bootstrap.min.css">
                        <script src="https://code.jquery.com/jquery-3.6.0.min.js"></script>
                        <script src="https://stackpath.bootstrapcdn.com/bootstrap/5.3.0/js/bootstrap.bundle.min.js"></script>

Cos'è un CDN?

Un CDN (Content Delivery Network) è una rete di server distribuiti geograficamente che forniscono contenuti web agli utenti in base alla loro posizione.

Utilizzare un CDN per Bootstrap offre vantaggi come maggiore velocità, affidabilità, caching, risparmio di banda e accesso a diverse versioni del framework.

A cosa serve il tag meta?

L'attributo <meta name="viewport" content="width=device-width, initial-scale=1"> serve a controllare il layout su dispositivi mobili.

width=device-width: Imposta la larghezza del viewport alla larghezza del dispositivo.

initial-scale=1: Imposta il livello di zoom iniziale a 1, evitando lo zoom automatico alla caricamento della pagina.

Differenza tra Container e Container Fluid

Il Container ha una larghezza fissa che si adatta alle diverse dimensioni dello schermo, mentre il Container Fluid occupa sempre il 100% della larghezza disponibile del viewport, rendendolo ideale per layout completamente responsive.
